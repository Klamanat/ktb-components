KTB Component Library

Web Components สำหรับระบบงาน KTB — สร้างด้วย StencilJS รองรับทุก Framework

StencilJS Web Components TypeScript Framework Agnostic

Installation

npm
npm install ktb-components
yarn
yarn add ktb-components

Usage

Via Script Tag (CDN)
<script type="module" src="https://cdn.jsdelivr.net/npm/ktb-components/dist/ktb-components.esm.js"></script>
React / Vue / Angular
import { defineCustomElements } from 'ktb-components/loader'; defineCustomElements();

Components

<ktb-alert>

Toast notification — รองรับ 4 ประเภท: success, error, warning, info

Live Preview
top-center

เลือกตำแหน่ง แล้วกด type ที่ต้องการ

Methods
Method Parameters Description
add() message: string, type?: AlertType, duration?: number เพิ่ม alert ใหม่ — duration หน่วย ms (default 3000, ใส่ 0 เพื่อไม่ให้ปิดอัตโนมัติ)
remove() id: number ปิด alert ตาม id พร้อม fade-out animation
AlertType
Value สี ไอคอน มี Title
success เขียว check-circle —
error แดง error-circle เกิดข้อผิดพลาด
warning ส้ม warning-triangle —
info น้ำเงิน info-circle —

HTML
<!-- วาง element ไว้ใน body --> <ktb-alert id="alert"></ktb-alert> <script> const alert = document.getElementById('alert'); alert.add('บันทึกสำเร็จ', 'success'); alert.add('เกิดข้อผิดพลาด', 'error', 0); // ไม่ปิดอัตโนมัติ </script>
React (JSX)
import { useRef } from 'react'; function App() { const alertRef = useRef(null); const showAlert = () => { alertRef.current?.add('บันทึกสำเร็จ', 'success'); }; return ( <> <ktb-alert ref={alertRef} /> <button onClick={showAlert}>Save</button> </> ); }
Vue
<template> <ktb-alert ref="alertRef" /> <button @click="showAlert">Save</button> </template> <script setup> import { ref } from 'vue'; const alertRef = ref(null); const showAlert = () => alertRef.value.$el.add('บันทึกสำเร็จ', 'success'); </script>
Angular
// app.module.ts — เพิ่ม CUSTOM_ELEMENTS_SCHEMA import { NgModule, CUSTOM_ELEMENTS_SCHEMA } from '@angular/core'; import { defineCustomElements } from 'ktb-components/loader'; defineCustomElements(); @NgModule({ schemas: [CUSTOM_ELEMENTS_SCHEMA], }) export class AppModule {}
// app.component.ts import { Component, ViewChild, ElementRef } from '@angular/core'; @Component({ template: ` <ktb-alert #alertRef></ktb-alert> <button (click)="showAlert()">Save</button> `, }) export class AppComponent { @ViewChild('alertRef') alertRef!: ElementRef<HTMLElement & { add: (msg: string, type: string, duration?: number) => Promise<void>; }>; showAlert() { this.alertRef.nativeElement.add('บันทึกสำเร็จ', 'success'); } }
