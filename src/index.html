KTB Component Library

Web Components สำหรับระบบงาน KTB — สร้างด้วย StencilJS รองรับทุก Framework

StencilJS Web Components TypeScript Framework Agnostic

Installation

npm
npm install ktb-components
yarn
yarn add ktb-components

Usage

Via Script Tag (CDN)
<script type="module" src="https://cdn.jsdelivr.net/npm/ktb-components/dist/ktb-components.esm.js"></script>
React / Vue / Angular
import { defineCustomElements } from 'ktb-components/loader'; defineCustomElements();

Components

<my-component>

แสดงชื่อจากส่วนประกอบ first, middle, last

Live Preview
Props
Property Attribute Description Type Default
first first ชื่อแรก (First name) string undefined
middle middle ชื่อกลาง (Middle name) string undefined
last last นามสกุล (Last name) string undefined
HTML
<!-- ใช้งานพื้นฐาน --> <my-component first="KTB" middle="Digital" last="Bank" ></my-component> <!-- ไม่มีชื่อกลาง --> <my-component first="Krungthai" last="Components"></my-component>
React (JSX)
import { defineCustomElements } from 'ktb-components/loader'; defineCustomElements(); export default function App() { return ( <my-component first="KTB" middle="Digital" last="Bank" /> ); }
Vue
<template> <my-component first="KTB" middle="Digital" last="Bank" /> </template> <script setup> import { defineCustomElements } from 'ktb-components/loader'; defineCustomElements(); </script>
