KTB Component Library

Web Components สำหรับระบบงาน KTB — สร้างด้วย StencilJS รองรับทุก Framework

StencilJS Web Components TypeScript Framework Agnostic

Installation

npm
npm install ktb-components
yarn
yarn add ktb-components

Usage

Via Script Tag (CDN)
<script type="module" src="https://cdn.jsdelivr.net/npm/ktb-components/dist/ktb-components.esm.js" ></script>
React / Vue / Angular
import { defineCustomElements } from 'ktb-components/loader'; defineCustomElements();

Design Tokens

Token ถูกแบ่งเป็น 2 ชั้น: Primitive (ค่าดิบ) และ Semantic (ความหมาย) — CSS custom properties ทะลุ Shadow DOM ได้ จึง override ได้จาก stylesheet ภายนอก

Token Architecture
/* Layer 1 — Primitive: raw values */ --primitive-blue-400: #60a5fa; --primitive-red-500: #ef4444; /* Layer 2 — Semantic: purpose-bound, reference primitives */ --color-primary: var(--primitive-blue-400); --color-danger: var(--primitive-red-500); /* Component tokens: override ผ่าน CSS selector ภายนอก */ ktb-button { --btn-primary-bg: var(--color-primary); }
